## Update rules

Update this page whenever:
- a decision is made
- a commitment is given
- a due date changes
- a preference or constraint becomes clear
- something goes wrong that should not be repeated

When updating, refresh:
- Last updated
- Next deliverable
- Next deliverable date
- Commitments ledger
- What not to do, if a new landmine appears

If a page section is blank, treat it as unknown. Do not invent context.
If current conversation conflicts with this page, current conversation wins.
